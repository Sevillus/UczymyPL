# Uczymy.pl
 Demo: https://uczymypl.vercel.app/
## About the project
Uczymy.pl is a website addressed to tutors. 
The main goal is to faciliate managment and lesson planning. 
# technologies
NextJs, ReactJs, Tailwind, mongoDb.
